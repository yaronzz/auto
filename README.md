## 🎨 使用

1. 安装 [autojs6.apk](https://github.com/SuperMonster003/AutoJs6)，使用说明搜索`autojs使用教程`
2. 下载项目下的所有文件到`autojs6`的脚本目录下
3. 配置`profile.json`文件

| 名称           | 说明              |
| -------------- | ----------------- |
| phone_password | 密码              |
| notify_sendkey | Server 酱(可不填) |
